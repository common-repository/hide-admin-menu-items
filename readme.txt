Hide Admin Menu Items provide a small functionality to hide rarely used menu items. Very simple !

== Description ==

*Hide Amin Menu Items* lets you manually edit the Dashboard menu. You can show / hide specific items.

== Installation ==

1. Download the zip file to your computer.
2. Unzip the file.
3. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
4. Activate the plugin through the 'Plugins' screen in WordPress
5. That's it. Go to the settings page and click to the *Hide items* sub-menu.
